Author Template Pack

Download the LaTeX template pack to prepare your submission. The zip contains everything you need:

arjournal.cls — the document class with all custom macros; include it alongside your .tex file

arjournal_template.tex — a comprehensive tutorial covering every feature: headings, inline formatting, quotations, footnotes, citations, figures, tables, and media embeds; read this file first

arjournal_references_template.bib — bibliography template with example entries for articles, books, chapters, theses, and online sources

transact_sample_article.pdf — a sample rendered article showing exactly how your submission will look once it is typeset and published; open this to preview the final output before you submit

When you are ready to submit, upload your .tex file and any media assets (images, audio, video) through the submission form. The platform runs the same LaTeX pipeline that generated the sample PDF, so what you see in that document is what your readers will see.
