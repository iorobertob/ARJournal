Media & Reference Files

What to upload and why

Files from your previous revision have been carried over automatically. You only need to upload files that are new or have changed. Uploading a file with the same filename as an existing one will replace it.

Bibliography file — references.bib (required)

Your BibTeX file. The platform parses it to render the bibliography with full author, title, and year. If you added or changed references, upload the updated .bib file — it must use the same filename referenced in your \bibliography{…} command.

Images — \ARJfigure{filename}{caption}{alt text}

Upload replacement images only if they changed. Any new \ARJfigure command you added to the revised manuscript needs its image file uploaded here. Filename must match exactly.

Video — \ARJvideo{filename}{caption}{poster}{transcript}

Upload new or updated video files, poster images, and transcripts. Supported formats: MP4, WebM, MOV.

Audio — \ARJaudio{filename}{caption}{transcript}

Upload new or updated audio files and transcripts. Supported formats: MP3, WAV, OGG, FLAC, AAC.
