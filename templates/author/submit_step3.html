Upload Media & Reference Files

What to upload and why

The platform matches each file you upload to the corresponding command in your .tex manuscript using the exact filename. Upload every file that your manuscript references.

Bibliography file — references.bib (required)

Your BibTeX file containing all cited references. The platform parses it to render the bibliography section with full author names, titles, and years. Without it, only cite keys (e.g. smith2024art) will appear. Upload whichever .bib filename your manuscript uses in \bibliography{…}.

Images — \ARJfigure{filename}{caption}{alt text}

Upload each image file referenced in your \ARJfigure commands. Supported formats: JPEG, PNG, WebP, GIF, TIFF. The filename in the macro must match the file you upload exactly (including extension and case).

\ARJfigure{photo1.jpg}{My caption}{Alt description}

Video — \ARJvideo{filename}{caption}{poster}{transcript}

Upload the video file and optionally a poster (thumbnail) image and a transcript file. All three filenames must match what you wrote in the macro. Supported video formats: MP4, WebM, MOV.

\ARJvideo{performance.mp4}{Caption}{poster.jpg}{transcript.pdf}

Audio — \ARJaudio{filename}{caption}{transcript}

Upload the audio file and optionally a transcript. Supported formats: MP3, WAV, OGG, FLAC, AAC.

\ARJaudio{soundwork.mp3}{Caption}{transcript.pdf}

Supplementary files — datasets, appendices, PDFs

Any additional files associated with your submission (datasets, code archives, extended appendices) can be uploaded here. They will be listed as supplementary materials.

Tip: If you upload a file with the same filename as one already listed, it will replace the previous version automatically.
